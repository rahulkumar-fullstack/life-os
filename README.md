# Life OS 🧠📒✅
### A Complete Vanilla JavaScript Project (HTML • CSS • JS)

**Life OS** is a real‑world, offline‑first **personal life management web application** built using **pure HTML, CSS, and modern JavaScript** — no frameworks, no libraries.

This project is designed as a **from‑scratch learning reference** that progressively covers **everything from web basics to advanced browser APIs**, with clean architecture, detailed comments, and explanations backed by **MDN Web Docs**.

---

## 🎯 Project Goals

- Build a **practical, day‑to‑day usable web app**
- Teach **modern web development from fundamentals to advanced**
- Maintain **production‑level code quality**
- Explain every concept in **simple, beginner‑friendly terms**
- Serve as a long‑term **learning & portfolio project**
- Stay **framework‑free** to deeply understand the web platform

---

## 🧩 What Is Life OS?

Life OS acts like a **personal operating system** for everyday needs:

- ✅ Task & Todo Management  
- 📝 Notes & Quick Ideas  
- 💰 Expense Tracking  
- 🔁 Habit Monitoring  
- 📅 Planning & Scheduling  
- ⚙️ User Preferences  
- 📡 Offline‑First Support  

All data lives **locally in the browser** and works **even without internet access**.

---

## 🛠️ Tech Stack (Intentionally Simple)

- **HTML5** – Semantic, accessible structure  
- **CSS3** – Modern layouts, variables, responsive design  
- **Vanilla JavaScript (ES6+)** – Clean, modular, modern JS  
- **Browser APIs** – LocalStorage, IndexedDB, Service Workers, etc.

> ❌ No React, ❌ No frameworks, ❌ No build tools  
> ✅ Just the **real web**
